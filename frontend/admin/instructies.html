Installeer de 2019-04-08 versie van raspbian. deze kan je downloaden met de torrent op https://afval.rubend.nl/raspbian.torrent.
Gebruik de internet kabel voor de installatie.
Log in met deze gegevens:
gebruikersnaampi
wachtwoordraspberry
Voer deze commands uit om de installatie te starten: 
wget https://afval.rubend.nl/setup
bash setup
De installatie duurt meestal niet langer dan 5 minuten op een raspberry pi 3b. Dit kan afwijken bij oudere versies van de pi.
